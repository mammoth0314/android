# 学生成绩管理系统

这是一个基于 Android 的单机版学生成绩管理系统示例，满足课程设计中常见的基础要求：

- 登录验证
- 记住账号
- 学生成绩列表展示
- 成绩新增、修改、删除
- 分数升序/降序排序
- SQLite 本地持久化
- Toast、Dialog、菜单、上下文菜单、Intent 传值

## 默认登录账号

- 账号：`admin`
- 密码：`123456`

## 主要页面

- `LoginActivity`：登录页
- `MainActivity`：成绩列表页
- `EditActivity`：新增/编辑页

## 说明

项目使用 Java + AndroidX + SQLite，实现了课程设计报告里提到的核心功能，可以直接放入 Android Studio 打开并同步运行。
